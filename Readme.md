## Install

`asdf` rocks:

```bash
asdf plugin-add rust https://github.com/code-lever/asdf-rust.git
asdf install rust 1.61.0
asdf global rust 1.61.0
```

## Run

`cd` into one of the directories and run:

```bash
rustc main.rs
chmod a+x main
./main
```
